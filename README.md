## 🎯 &nbsp;À propos du repo

Ce dépôt est mon **terrain d'entraînement personnel** — un espace où je reproduis le format des piscines Epitech pour m'exercer sur de **nouveaux langages de programmation**, tout en conservant une trace de mon parcours à la piscine C officielle.

> *Chaque piscine est une guerre contre l'inconnu. Ce repo est l'armurerie.*

Le principe est simple : prendre le format intensif, exigeant et formatif des piscines Epitech, et l'appliquer à n'importe quel langage que je souhaite maîtriser. On ne subit pas les langages — on les dompte.

---
